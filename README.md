
# Steam Wishlist Helper

结合ITAD的价格查询功能, 生成一份自己的游戏折扣列表

需要Python3.8及以上

## 使用方法

* 从[发布页][0]下载最新版本并解压
* pip install -r requirements.txt
* 将 example.config.toml 重命名为 config.toml
* 修改 config.toml , 填入自己的API Token([申请地址][1])
* python3 run.py
